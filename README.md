  После копирования проекта и выполнения миграции БД TimesheetDB в PostgreSQL:
  1. api/user/authenticate - вводим логин/пароль из списка, входим данные, получаем токен.
  2. Кнопка "Authorize" - вводим "Bearer _наш_токен", входим в систему.

  Пользователи Employee и Customer имеют доступ только к своей части и не могут смотреть системную информацию (данные пользователей и др.).
